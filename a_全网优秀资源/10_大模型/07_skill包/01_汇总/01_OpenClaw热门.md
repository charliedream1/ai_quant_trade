以下为社区实测安装量最高、覆盖**投研/量化/风控**全链路的金融垂直技能，附核心能力与安装入口，可直接复制部署。

1、行情与数据类（基础必备）

（1） stock-analysis
核心：**个股**8维诊断（基本面/技术面/资金面/情绪面）、财报解读、谣言检测，支持A股/港股/美股，安装量623+。
入口： clawhub install stock-analysis

（2） tushare-finance
核心：**A股**220+全量财务接口，覆盖财报、分红、解禁、K线数据，为国内投研数据底座 。
入口： clawhub install tushare-finance

（3） stock-watcher
核心：**自选股**实时监控，异动（涨跌幅/成交量/公告）自动推送，堪称“24小时不打烊盯盘助手”。
入口： clawhub install stock-watcher

（4） us-stock-analysis
核心：**美股**深度分析，含财报电话会议纪要、机构评级、估值测算，覆盖NVDA等美股核心标的。
入口： clawhub install us-stock-analysis

2、量化与策略类（核心工具）

（1） finlab
核心：**一站式量化**分析平台，内置策略模板、因子计算、组合优化，安装量515+，适合新手搭建量化框架。
入口： clawhub install finlab

（2） tradingview-quantitative
核心：TradingView指标对接，支持100+**技术指标**（MACD/RSI/布林带）、形态识别，直接用于技术面选股。
入口： clawhub install tradingview-quantitative

（3） backtesting
核心：**历史回测**引擎，输出收益率、最大回撤、夏普比率，快速验证选股/交易策略有效性。
入口： clawhub install backtesting

（4） stock-market-pro
核心：专业**股市分析**套件，含多因子模型、市场情绪指数、行业轮动分析，进阶投研必备。
入口： clawhub install stock-market-pro

3、资讯与合规类（风控辅助）

（1） policymonitor
核心：央行/证监会/发改委政策实时巡检，新政出台自动推送，捕捉宏观政策对市场的影响。
入口： clawhub install policymonitor

（2） skillvetter
核心：第三方技能安全检测，拦截权限请求、数据外发等风险，为金融数据安全加“第一道门”。
入口： clawhub install skillvetter

（3） multi-search-engine
核心：多引擎（Google/Bing/DuckDuckGo）并行搜索，财经新闻、行业研报一键聚合，提升信息获取效率。
入口： clawhub install multi-search-engine

4、辅助效率类（提效神器）

（1) office-document-specialist-suite
核心：**金融报告自动化**，支持Excel数据处理、Word研报排版、PPT结构化，直接输出可交付成果。
入口： clawhub install office-document-specialist-suite

(2) agent-browser
核心：**浏览器自动化**，抓取网页端财经数据、表单录入、公告检索，解放重复操作。
入口： clawhub install agent-browser

快速上手建议

- 优先装stock-analysis + tushare-finance + stock-watcher，搞定“数据-分析-监控”基础链路；
- 想做量化可加finlab + backtesting，快速验证策略；
- 合规敏感场景必装skillvetter + policymonitor，筑牢安全与信息边界。
