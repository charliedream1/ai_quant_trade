# 1. stock_prediction 

- Github (323 stars): https://github.com/KittenCN/stock_prediction

一个基于 PyTorch 的股价预测项目，涵盖从数据收集、预处理、模型训练到推理的整个工作流程。该项目侧重于多尺度混合模型（TemporalHybridNet）和PTFT + Variational SSM组合。

特征
- 市场数据收集：支持 tushare、akshare 和 yfinance 作为数据源。
- 数据预处理：将每日CSV聚合到，支持重复加载。pkl_handle/train.pkl
- 特征工程：自动生成日志收益、差异、宏观/行业/情绪外生变量，支持按符号归一化和股票嵌入。
- 模型训练：统一入口，支持 LSTM、Transformer、TemporalHybridNet、PTFT_VSSM、Diffusion、Graph 等，内置训练器和提前停止。
- 推理：与训练中一致的特征处理和嵌入，输出预测图表和指标。
- 评估指标：自动收集 RMSE、MAPE、分位数覆盖率、VaR、CVaR 等，并将结果保存到 .output/metrics_*.json
- 技术指标：内置MACD、KDJ、DMI、ATR等。
